# aws-high-availability-architecture
secure and scalalble AWS architecture with auto sacling and load  Balancing 
This project demonstrates a secure and scalable AWS cloud architecture designed for high availability.

🏗️ Architecture
VPC with public and private subnets
Bastion Host
EC2 instances in private subnet
Application Load Balancer
Auto Scaling Group
⚙️ Services Used

EC2, VPC, IAM, ALB, Auto Scaling, CloudWatch

🔐 Security
Private subnet deployment
IAM roles
Security groups
📊 Monitoring
CloudWatch metrics
Alerts configured
🔄 Auto-Healing
Instance failure simulated
Auto Scaling replaced unhealthy instance
